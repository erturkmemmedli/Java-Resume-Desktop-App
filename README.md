# Java Resume Desktop App

Key words for this project:
- Object Oriented Programming (OOP) - Java
- Desktop (GUI) Programming - Swing
- Datbase Connection - JDBC
- Database - MySQL
- JAVA IDE software - IntelliJ IDEA Community Edition 2022.2.2
- MySQL software - Navicat Premium 16

This app is designed to demonstrate resume of users in database. Furthermore, new resumes can be added to the database, existing resumes can be updated and deleted. All necessary sections of CV (general information, contact information, experience and skills) can easily be modified, added, and deleted. Overall project is designed based on OOP standards and SOLID principles.
